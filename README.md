# Superstore-sales-data-analysis-and-forecasting

## Overview

This project analyzes retail sales data to understand business trends and predict future sales. Using Python and machine learning, the goal is to explore how factors like region, category, quantity, and discount influence sales performance.

The project combines data analysis, visualization, and a simple machine learning model to demonstrate practical data science skills.

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn

## What This Project Includes

* Exploratory data analysis of sales data
* Visualizations to understand sales patterns
* Analysis of sales by region and category
* Monthly sales trend analysis
* A Random Forest machine learning model to predict sales
* Feature importance analysis to understand key factors affecting sales

## Dataset

The project uses the **Sample Superstore dataset**, which contains retail sales records including product category, region, quantity, discount, and sales information.

## Key Insights

* Some product categories contribute significantly more to total sales.
* Certain regions consistently generate higher revenue.
